Persyaratan Pendaftaran

I. KEUANGAN

  • 1. Membayar Uang Pendaftaran gelombang 1 sebesar        : Rp. 200.000,-
  • 2. Membayar Uang Pendaftaran gelombang 2 sebesar        : Rp. 250.000,-

II. KELENGKAPAN ADMINISTRASI

  • Untuk lulusan tahun 2019 kebawah :
    Foto Copy Legalisr Ijasah SMA/SMK/MAN + SKHUN        : 3 lembar
  • Untuk lulusan tahun 2020 :
    1. Foto Copy Legalisr Ijasah SMA/SMK/MAN + SKHUN          : 3 lembar
    2. Pas Foto ukuran 3x4 cm terbaru ( berpakaian rapih)        : 2 Lembar
    3. Foto Copy KTP ( wajib dikumpulkan ) sebanyak                 : 2 Lembar
    4. Foto copy Kartu Keluarga( wajib dikumpulkan )sebanyak : 2 Lembar
    5. Mengisi Formulir dan Mengembalikan ke Petugas
    6. Menyerahkan Map SnelHekter plastick sebanyak             : 1 Buah
  • Dengan masing-masing warna :
    A. Fakultas Keguruan dan Ilmu Pendidikan berwarna     : BIRU
    1. Program Studi Pendidikan Guru Sekolah Dasar
    2. Program Studi Pendidikan Bahasa dan Sastra Indonesia
    3. Program Studi Pendidikan Matematika
    4. Program Studi Pendidikan Sejarah
    5. Program Studi Pendidikan Fisika
    6. Program Studi Pendidikan Ekonomi
    7. Program Studi Pendidikan Biologi

    B. Fakultas Bahasa dan Sastra berwarna  : ORANGE
    1. Program Studi Sastra Inggris

    C. Fakultas Ekonomi berwarna  : KUNING
    1. Program Studi Manajemen
    2. Program Studi Akuntansi S1
    3. Program Studi Ekonomi Pembangunan

    D. Fakultas Pertanian berwarna  : HITAM
    1. Program Studi Agroteknologi

    E. Fakultas Teknik berwarna  : MERAH
    1. Program Studi Teknik Sipil
    2. Program Studi Arsitektur

    F. Fakultas Hukum berwarna  : HIJAU
    1. Program Studi Ilmu Hukum

    G. Fakultas Teknologi Informsi berwarna  : ORANGE
    1. Program Studi Sistem Informasi

CATATAN

  • Pendaftaran dengan sistem 2 pilihan Program Studi : Pilihan 1 dan pilihan 2 Contoh : Pilihan 1 : Prodi Manajemen dan Pilihan 2 : Prodi Pendidikan Sejarah.Hasil kelulusan ada 3 kelompok yaitu : Lulus pada pilihan 1 atau lulus pada pilihan 2 atau tidak lulus,dilihat dari hasil testing dan Nilai Hasil Ujian Sekolah dan atau SKHUN dengan jumlah kuota yang diterima pada masing-masing program studi yang ada.

  • Bagi yang belum melengkapi persyaratan pendaftaran ( LEGALISIR IJASAH, KK, KTP) wajib membawa materai 6000 untuk membuat surat pernyataan.

  • Bagi yang mendaftar dengan sistem online wajib menuliskan No. Hp, dan No Hp diharapkan tidak boleh diganti karena ada info lebih lanjut yang akan diinformasikan melalui no hp masing-masing.

  • Bagi yang mendaftar dengan sistem online persyaratan kelengkapan wajib dibawa saat mengikuti testing dan wajib membawa dan menunjukkan KTP asli.
